## 📊 컨텐츠 분량 목표

### 벤치마크 분석

#### 유사 게임 플레이 타임
| 게임 | 1회 런 | 총 플레이 타임 |
|:---|:---:|:---:|
| **Brotato** | 15-25분 | 20-30시간 |
| **Vampire Survivors** | 15-30분 | 30-50시간 |
| **Survivor.io** | 10-20분 | 무한 (F2P) |
| **Magic Survival** | 15-25분 | 20-40시간 |

### 목표 플레이 타임

#### 1회 플레이 (런)
```
목표: 15-20분
- 웨이브당 평균 1분
- 상점 시간 포함
- 총 15-20 웨이브
```

**선정 이유**:
- ✅ 모바일 환경에 적합 (출퇴근, 쉬는 시간)
- ✅ 집중력 유지 가능
- ✅ "한 판 더" 심리 유발
- ✅ 중단해도 부담 없는 길이

#### 총 컨텐츠 분량

| 단계 | 플레이 타임 | 목표 |
|:---|:---:|:---|
| **MVP** | 2-3시간 | 게임 이해 + 1회 클리어 |
| **출시 버전** | 10-15시간 | 다양한 빌드 실험 |
| **장기 운영** | 30-50시간+ | 메타 진행 + 컬렉션 |

## 🚀 Phase 1: MVP (최소 기능 제품)

### 개발 기간
- **목표**: 2-3개월
- **마일스톤**: 플레이 가능한 프로토타입

### 핵심 목표
> "재미있는 코어 루프를 증명한다"

**코어 루프**:
```
전투 → 경험치/골드 획득 → 레벨업/상점 → 강해짐 → 더 어려운 적 → 반복
```

### 필수 구현 목록

#### ✅ 전투 시스템
- [ ] 플레이어 이동 (조이스틱)
- [ ] 무기 자동 조준/공격
- [ ] 적 AI (추격, 공격)
- [ ] 데미지 계산 (치명타 포함)
- [ ] 체력/경험치 시스템

#### ✅ 웨이브 시스템
- [ ] 웨이브 진행 (12-15개)
- [ ] 웨이브별 난이도 증가
- [ ] 적 스폰 시스템
- [ ] 보스 웨이브 (1개)

#### ✅ 아이템 시스템
**무기**: 10종
- 근접 무기 3종 (검, 단검, 도끼)
- 원거리 무기 2종 (활, 지팡이)
- 등급: Common (5종), Uncommon (5종)
- **특수 효과 제외** (기본 공격만)

**장신구**: 15종
- 스탯 변경만 구현
- 등급: Common (8종), Uncommon (7종)
- **특수 효과 제외**

**Trade-off 시스템**:
- 모든 아이템에 긍정 + 부정 효과
- 스탯 변경 시스템 완전 구현

#### ✅ 상점 시스템
- [ ] 웨이브 종료 후 상점 오픈
- [ ] 아이템 4개 랜덤 진열
- [ ] 구매 시 골드 차감 + 스탯 적용
- [ ] 재롤 기능 (골드 소모)
- [ ] 무기 슬롯 제한 (6개)

#### ✅ 드롭 시스템
- [ ] 골드 드롭
- [ ] 경험치 드롭
- [ ] 체력 회복 드롭 (기본)

#### ✅ 적 시스템
- [ ] 기본형 적 (근접)
- [ ] 원거리형 적
- [ ] 빠른형 적
- [ ] 보스 1종

#### ✅ UI/UX
- [ ] 체력/경험치 바
- [ ] 골드 표시
- [ ] 웨이브 진행 표시
- [ ] 무기 인벤토리 UI
- [ ] 상점 UI
- [ ] 아이템 툴팁 (긍정/부정 효과)

### 제외 항목 (MVP에서 구현하지 않음)

#### ❌ 고급 아이템
- Rare/Legendary 아이템
- 무기 특수 효과 (관통, 폭발, 체인 등)
- 장신구 특수 효과 (이중 타격, 흡혈 등)

#### ❌ 메타 진행
- 캐릭터 선택
- 영구 업그레이드
- 업적/도전 과제

#### ❌ 추가 컨텐츠
- 다양한 스테이지/환경
- 엔드게임 콘텐츠
- 스토리/튜토리얼

### MVP 컨텐츠 분량

| 요소 | 수량 |
|:---|:---:|
| **웨이브 수** | 12-15개 |
| **1회 플레이 시간** | 12-15분 |
| **무기 종류** | 10종 |
| **장신구 종류** | 15종 |
| **적 타입** | 3종 + 보스 1종 |
| **캐릭터** | 1개 (기본) |
| **총 플레이 타임** | 2-3시간 |

### 성공 지표
- [ ] 코어 루프가 재미있다 (플레이테스트)
- [ ] 10-15분 플레이로 클리어 가능
- [ ] 아이템 선택이 의미있다
- [ ] 다시 하고 싶은 동기 부여

## 🎮 Phase 2: 출시 버전 (Launch)

### 개발 기간
- **목표**: MVP 이후 2-3개월
- **마일스톤**: 스토어 출시

### 핵심 목표
> "다양성과 리플레이성을 확보한다"

### 추가 구현 목록

#### ✅ 아이템 확장
**무기**: 20종 추가 (총 30종)
- 근접 무기 5종 추가
- 원거리 무기 5종 추가
- Rare 등급 8종
- Legendary 등급 7종

**무기 특수 효과**:
- Uncommon: 관통, 분산, 회전
- Rare: 폭발, 연쇄, 출혈
- Legendary: 전기 충격, 유도탄, 무한 관통

**장신구**: 25종 추가 (총 40종)
- Rare 등급 12종
- Legendary 등급 13종

**특수 효과 장신구**: 8-10종
- 가시 갑옷 (반사)
- 회복의 목걸이 (지속 회복)
- 이중 타격 (확률 2회 공격)
- 불사조의 깃털 (부활) 등

#### ✅ 캐릭터 시스템
- [ ] 캐릭터 3-5종
- [ ] 각 캐릭터 고유 시작 스탯
- [ ] 각 캐릭터 시작 무기 다름
- [ ] 캐릭터 선택 UI

**캐릭터 예시**:
1. **전사**: 높은 체력, 근접 특화
2. **궁수**: 높은 사거리, 원거리 특화
3. **도적**: 빠른 이동속도, 크리티컬 특화
4. **마법사**: 높은 공격력, 낮은 체력

#### ✅ 메타 진행
- [ ] 영구 업그레이드 시스템
- [ ] 게임 재화 (크리스탈 등)
- [ ] 업그레이드 5-8종

**업그레이드 예시**:
- 시작 골드 +20
- 시작 체력 +10
- 경험치 획득량 +10%
- 상점 아이템 수 +1
- 재롤 비용 -20%

#### ✅ 적 확장
- [ ] 적 타입 5종 추가 (총 8-10종)
- [ ] 보스 2종 추가 (총 3종)
- [ ] 적 특수 패턴 (돌진, 분열 등)

#### ✅ 웨이브 확장
- [ ] 웨이브 20-25개로 확장
- [ ] 난이도 곡선 조정
- [ ] 보스 웨이브 3개

#### ✅ 추가 UI/UX
- [ ] 캐릭터 선택 화면
- [ ] 메타 업그레이드 화면
- [ ] 컬렉션 도감
- [ ] 설정/옵션 메뉴
- [ ] 사운드/BGM

#### ✅ 튜토리얼
- [ ] 기본 조작 가이드
- [ ] 상점 사용법
- [ ] 스탯 시스템 설명

### 출시 버전 컨텐츠 분량

| 요소 | 최소 | 권장 | 이상적 |
|:---|:---:|:---:|:---:|
| **웨이브 수** | 15개 | 20개 | 25개 |
| **1회 플레이 시간** | 12분 | 15-20분 | 25분 |
| **무기 종류** | 20종 | 25-30종 | 35종 |
| **장신구 종류** | 25종 | 35-40종 | 50종 |
| **캐릭터 수** | 3개 | 4-5개 | 6-8개 |
| **적 타입** | 6종 | 8-10종 | 12종 |
| **영구 업그레이드** | 5종 | 6-8종 | 10종 |
| **총 플레이 타임** | 8시간 | 10-15시간 | 20시간 |

### 성공 지표
- [ ] 10시간 이상 플레이 가능한 컨텐츠
- [ ] 최소 3가지 유효한 빌드 전략
- [ ] 캐릭터별 차별화된 플레이 경험
- [ ] 리텐션 D1 > 40%, D7 > 20%

## 📈 Phase 3: 장기 운영 (Live Service)

### 개발 기간
- **목표**: 출시 후 지속 업데이트
- **주기**: 월 1-2회 업데이트

### 핵심 목표
> "플레이어를 유지하고 커뮤니티를 성장시킨다"

### 업데이트 계획

#### 정기 컨텐츠 (월간)
- [ ] 신규 캐릭터 1-2개
- [ ] 신규 아이템 세트 5-10종
- [ ] 신규 적 타입 1-2종
- [ ] 밸런스 패치

#### 시즌/이벤트 (분기)
- [ ] 시즌 테마 (크리스마스, 할로윈 등)
- [ ] 기간 한정 아이템
- [ ] 특별 도전 과제
- [ ] 랭킹 시스템

#### 확장 시스템
- [ ] **아이템 진화**: 동일 아이템 중복 획득 시 업그레이드
- [ ] **세트 아이템**: 특정 조합 시 추가 효과
- [ ] **저주받은 아이템**: 강력한 효과 + 심각한 패널티
- [ ] **아이템 합성**: 여러 아이템을 합쳐 새 아이템 생성

#### 엔드게임 컨텐츠
- [ ] 무한 모드 (웨이브 무한 증가)
- [ ] 타임 어택 모드
- [ ] 데일리 챌린지
- [ ] 보스 러시 모드

#### 소셜 기능
- [ ] 친구 시스템
- [ ] 리더보드
- [ ] 빌드 공유 기능
- [ ] 리플레이 저장/공유

### 장기 운영 목표

| 기간 | 컨텐츠 목표 | 플레이 타임 |
|:---|:---|:---:|
| **출시 후 3개월** | 캐릭터 8개, 아이템 80종 | 20-25시간 |
| **출시 후 6개월** | 캐릭터 12개, 아이템 120종 | 30-40시간 |
| **출시 후 1년** | 캐릭터 15개+, 아이템 150종+ | 50시간+ |

## 🛠️ 개발 리소스 계획

### 1인 개발 시나리오

#### Phase 1 (MVP): 3개월
```
월 1: 코어 시스템 (전투, 이동, 스탯)
월 2: 아이템 시스템, 상점, 드롭
월 3: 웨이브 시스템, 적 AI, 밸런싱
```

#### Phase 2 (출시): 추가 3-4개월
```
월 4: 아이템 확장 (30종 → 완성)
월 5: 캐릭터 시스템, 메타 진행
월 6: 적 확장, 보스 추가
월 7: UI/UX 완성, 튜토리얼, 테스트
```

**총 개발 기간**: 6-7개월

### 소규모 팀 (2-3명) 시나리오

#### Phase 1 (MVP): 2개월
```
월 1: 코어 시스템 구현
월 2: 컨텐츠 제작 + 밸런싱
```

#### Phase 2 (출시): 추가 2-3개월
```
월 3: 아이템/캐릭터 확장
월 4: 메타 시스템 + 적 확장
월 5: 퀄리티 향상 + 출시 준비
```

**총 개발 기간**: 4-5개월

### 필요 리소스

#### 프로그래밍
- 코어 시스템: 40%
- 아이템 시스템: 20%
- UI/UX: 15%
- 적 AI: 15%
- 기타: 10%

#### 아트 (픽셀 아트)
- 플레이어 캐릭터: 5종 (애니메이션 포함)
- 무기 스프라이트: 30종
- 적 캐릭터: 10종 (애니메이션 포함)
- 이펙트: 공격, 피격, 드롭 등
- UI 아트: 상점, 인벤토리, 아이콘 등

#### 사운드
- BGM: 3-5곡 (메인, 전투, 보스 등)
- SFX: 20-30종 (공격, 피격, UI 등)

## ⚠️ 리스크 관리

### 개발 리스크

#### 스코프 크립 (Scope Creep)
**위험**: 기능 계속 추가 → 개발 지연
**대응**:
- MVP 범위 엄격히 준수
- "Nice to have" 기능은 Phase 2로 연기
- 매주 범위 점검

#### 밸런싱 지옥
**위험**: 아이템 30종+ 밸런싱 불가능
**대응**:
- MVP에서 10-15종으로 밸런싱 검증
- 스프레드시트로 수치 관리
- 자동화 테스트 도구 개발

#### 컨텐츠 부족
**위험**: 2-3시간 만에 할 게 없어짐
**대응**:
- 메타 진행 시스템 필수
- 캐릭터별 차별화
- 리플레이 가치 극대화 (랜덤성)

#### 기술적 난이도
**위험**: 특수 효과 구현 실패
**대응**:
- MVP에서는 특수 효과 제외
- 단계적 구현 (관통 → 폭발 → 유도탄)
- 레퍼런스 프로젝트 참고

### 시장 리스크

#### 경쟁 게임
**위험**: 유사 게임 포화 상태
**대응**:
- 차별화 요소 강조 (16비트 픽셀 아트)
- 독특한 아이템 효과
- 빠른 출시로 선점

#### 플레이어 리텐션
**위험**: 초반 이탈율 높음
**대응**:
- 온보딩 최적화
- 초반 성취감 제공
- 명확한 진행 목표

## 🎯 핵심 원칙

### 개발 철학

#### 1. **코어 루프 우선**
> 화려한 기능보다 반복해도 재미있는 핵심 게임플레이

#### 2. **빠른 프로토타입**
> 3개월 내 플레이 가능한 버전 완성

#### 3. **데이터 기반 밸런싱**
> 감이 아닌 플레이테스트 데이터로 조정

#### 4. **점진적 확장**
> MVP → 출시 → 운영 순서로 단계적 발전

#### 5. **플레이어 피드백 중심**
> 출시 후 데이터와 피드백으로 방향 전환

### 성공 기준

#### MVP
- [ ] 15분 플레이가 재미있다
- [ ] 다시 하고 싶다
- [ ] 아이템 선택이 의미있다

#### 출시 버전
- [ ] 10시간 이상 플레이 가치
- [ ] 다양한 빌드 전략
- [ ] 리텐션 D1 > 40%

#### 장기 운영
- [ ] MAU 10,000+
- [ ] 평균 세션 시간 20분+
- [ ] 커뮤니티 활성화
